Traditional TV Equipment

Traditional TV Equipment uses rows of phosphor dots that would light up when the dots recieved energy. Each set of dots had the primary colours of light, and by using different amounts each coloured light would help make a color. The surrounding dots also help cause this effect. For example, if a red dot is lit up and nearby green dots are lit up you'll see it as yellow.
